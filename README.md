# futo

タイム・ペースを表示せず、走った道が「育つ」ランニングアプリ

## コンセプト

futo は、従来のランニングアプリのようにタイムやペースを前面に出すのではなく、走った軌跡が地図上で「育って」いく体験を提供します。数字に縛られず、走ること自体を楽しめるアプリを目指しています。

## 技術スタック

- **React Native** (Expo)
- **TypeScript**
- **expo-location** … GPS 取得
- **react-native-maps** … 地図表示

## 実装済み機能

- GPS 取得
- 地図表示
- GPS と地図の連動（現在地の自動追従）

## 今後の予定

- データ保存
- ヒートマップ表示

## セットアップ方法

```bash
npm install
npx expo start
```

iOS/Android 実機やシミュレータでアプリを起動できます。位置情報の権限を許可してください。
